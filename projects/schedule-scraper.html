Schedule Scraper Go to Github
Export RoSI/ACORN timetables to iCal format.

JavaScript JQuery

Generate an .ical file from your RoSI or ACORN timetable. The iCal file can then be easily imported into your application of choice, such as Google Calendar.

Usage

Three super easy ways to use this script!

Bookmarklet (recommended)

Run the script from a bookmark. Simple and reusable!

  1. Drag and drop Schedule scraper into your Bookmarks bar.
    Chrome: You may need to push ctrl + shift + B to open your Bookmarks bar.
  2. Navigate to the timetable to be exported.
  3. Click the bookmark!

Run in address bar

  1. Navigate to the timetable to be exported.
  2. Paste the following script into your address bar. 
    javascript:$(function(){$.getScript('https://gitcdn.xyz/repo/Shadowen/ScheduleScraper/master/ScheduleScraper.js')});
  3. Prepend the script with javascript: if it's not already there.
  4. Hit enter!

Run in JavaScript console

  1. Navigate to the timetable to be exported.
  2. Open the JavaScript console with ctrl+shift+J.
  3. Paste the following script into the console. 
    $(function(){$.getScript('https://gitcdn.xyz/repo/Shadowen/ScheduleScraper/master/ScheduleScraper.js')});
  4. Press enter!
