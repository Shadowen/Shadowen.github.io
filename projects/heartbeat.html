A custom Java-Arduino serial bridge

Java Arduino

The Heartbeat serial monitor program was a custom serial monitor written to work with a matching library on the Arduino. Heartbeat can be used as both a debugging interface and an I/O method for the robot. It offers extended capabilities to monitor various variables on the Arduino from a computer.

The RXTX library was used to allow the computer to receive serial communication in Java, which allowed custom graphics and user interface to easily be drawn on the screen.

This library was mainly designed for use in the AER201 design competition alongside the robot.
